Google Translate Widget makes your website multilingual and available to the world using Google Translate

== Description ==

Google Translate Widget aka [GTranslate](http://gtranslate.net/?xyz=1108) is a multilingual website solution for WordPress.

**Watch GTranslate Tour**
[vimeo http://vimeo.com/30132555]

With Google Translate Widget you can make your website multilingual easily.

Please `use` [GTranslate Forum](http://gtranslate.net/forum/) for your questions and support requests!

== Installation ==

1. Upload `gtw` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can add a widget to your website or use [gtrans] syntax inside pages where you want it to appear.

Find installation manual on our website: [GTranslate Documentation](http://gtranslate.net/docs)

== Frequently Asked Questions ==

= What is the quality of translation? =

It's Google translation, hence Google quality. We are using Google Translate free service. In the Pro and Enterprise versions you can refine the translations manually and order professional translations directly from your site.

= Which languages are supported? =

Here is the list: Afrikaans, Albanian, Arabic, Armenian, Azerbaijani, Basque, Belarusian, Bulgarian, Catalan, Chinese (Simplified), Chinese (Traditional), Croatian, Czech, Danish, Dutch, English, Estonian, Filipino, Finnish, French, Galician, Georgian, German, Greek, Haitian Creole, Hebrew, Hindi, Hungarian, Icelandic, Indonesian, Irish, Italian, Japanese, Korean, Latvian, Lithuanian, Macedonian, Malay, Maltese, Norwegian, Persian, Polish, Portuguese, Romanian, Russian, Serbian, Slovak, Slovenian, Spanish, Swahili, Swedish, Thai, Turkish, Ukrainian, Urdu, Vietnamese, Welsh, Yiddish

= Can I exclude some parts from being translated? =

It is available in commercial versions only. Please visit our website to find out more. [GTranslate](http://gtranslate.net/?xyz=1108)

= When will you be adding support for new languages? =

Since the widget only uses Google Translate, we have no control of when new languages will be available. Once Google adds a new language it will be available automatically.

= Google Translate added a new language, and there is no flag for it yet, what to do? =

Please contact us and let us know, so we can add the appropriate flag for you quickly.

== Configuration ==

To configure the widget go to Settings -> gtrans
